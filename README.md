Mini project mata kuliah **Algoritma dan Pemrograman** yang mengimplementasikan:
- **Divide-Conquer-Combine** (Branching Factor, Balance, Data Dependence, Parallelism, Sequential)
- **Sorting** (Bubble, Selection, Insertion, Merge, Quick, Counting, Radix)
- **Searching** (Sequential, Hash Table, BST, Binary, Interpolation, Jump)
- **Greedy** (Activity Selection, Fractional Knapsack, Dijkstra)
